## Question 1

What is the relation between Git and GitHub?

- [ ] GitHub is a main section in the Git project workflow.
- [x] GitHub is an online hosting service for Git repositories.
- [ ] GitHub is a program required for version control on Git.
- [ ] GitHub is an open-source tool used to create Git.

**Answer:** GitHub is an online hosting service for Git repositories.

**Explanation:** Correct! GitHub is one of the most popular web-hosted services for Git repositories.

**Score:** 1 / 1 point

## Question 2

Which of the following describes a repository?

- [ ] A collection of user accounts that are connected to a pull request
- [x] A data structure that stores documents and source code
- [ ] A basic unit of storage in Git that cannot be changed
- [ ] A plain text file that explains the purpose of a project

**Answer:** A data structure that stores documents and source code

**Explanation:** Correct! A repository is a data structure for storing documents, including application source code.

**Score:** 1 / 1 point

What is the purpose of a license?

- [x] To express how people can use your code
- [ ] To establish a secure remote login between computers
- [ ] To describe the purpose of the project
- [ ] To record all changes made to a repository

**Answer:** To express how people can use your code

**Explanation:** Correct! The purpose of a license file is to express how people can use your code.

## Question 4

What is the role of “commit” in GitHub?

- [ ] To undo changes
- [ ] To pull requests
- [ ] To merge
- [x] To save changes

**Answer:** To save changes

**Explanation:** Correct! To save your changes to a GitHub repository, you must commit the changes.

## Question 5

What type of version control system is Git?

- [ ] Remote
- [ ] Centralized
- [x] Distributed
- [ ] Local

**Answer:** Distributed

**Explanation:** Correct! Git is a distributed version control system, meaning each team member has a copy of the entire project, including its files and revision history, on their computer.

**Score:** 1 / 1 point

## Question 6

Which of the following statements is associated with a working directory?

- [x] Contains the files and subdirectories on the computer that are associated with a Git repository
- [ ] Contains project folders that are set up for version control
- [ ] Copy of a repository
- [ ] A separate line of development that allows one to work independently

**Answer:** Contains the files and subdirectories on the computer that are associated with a Git repository

**Explanation:** Correct! A working directory contains the files and subdirectories on your computer that are associated with a Git repository.

## Question 7

What happens if you make a change on a repository branch that you don't own?

- [ ] GitHub undoes the changes.
- [ ] GitHub issues a push request for the owner.
- [x] GitHub makes a pull request
- [ ] GitHub generates an error message.

**Answer:** GitHub makes a pull request

**Explanation:** Correct! GitHub automatically makes a pull request on your behalf if you make a change on a repository branch that you do not own.
