## Question 1

Which of the following does the `main` branch store?

- [X] The deployable version of the project code
- [ ] The original and unchanged versions of the code
- [ ] The files that contain documentation only
- [ ] The project files, including files from forks

**Answer:** The deployable version of the project code

**Explanation:** Correct! The `main` branch is definitive by default, stores the deployable code, and is the official working version of your project.

**Score:** 1 / 1 point

## Question 2

Which of the following best describes the function of a pull request?

- [X] Helps to merge the child branch with the `main` branch
- [ ] Helps to create a copy of the `main` branch
- [ ] Shows the difference in the content between the child branch and the `main` branch
- [ ] Facilitates the creation of a child branch

**Answer:** Helps to merge the child branch with the `main` branch

**Explanation:** Correct. A pull request does help to merge the child branch with the `main` branch.

**Score:** 1 / 1 point

## Question 3

Identify the correct statement.

- [ ] To add a file to the child branch, you need to add it to the `main` branch first.
- [ ] If you add a file to the child branch, it will automatically reflect in the `main` branch.
- [X] You can add files in the child branch without adding any to the `main` branch.
- [ ] It is not possible to add a file to the child branch.

**Answer:** You can add files in the child branch without adding any to the `main` branch.

**Explanation:** Correct! You can add files in the child branch without adding any to the `main` branch. If you add a file to the child branch, the file does not get added to the `main` branch on its own.

**Score:** 1 / 1 point
