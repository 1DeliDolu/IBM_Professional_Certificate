## Practice Quiz — Informational, Navigational, & Management Commands

### Question 1

A Linux shell is an interface for running commands. It is also:

- [ ] A user-defined algorithm.
- [ ] An operating system.
- [ ] an error-logging script.
- [x] An interactive language.

**Answer:** An interactive language. A shell enables access to files, utilities, and applications and can be used as an interactive scripting language to automate tasks.

**Score:** 1 / 1

### Question 2

Which of the following commands could you use to list running processes and information about each process? (Select two.)

- [ ] `man`
- [ ] `echo`
- [x] `top` — shows running processes and resource usage (like Task Manager).
- [x] `ps` — displays currently running processes and runtime information.

**Answers:** `top`, `ps`.

**Score:** 1 / 1

### Question 3

Which of the following common shell commands for navigating files and directories prints the current working directory?

- [ ] `cd`
- [ ] `ls`
- [x] `pwd`
- [ ] `find`

**Answer:** `pwd` — prints the present working directory.

**Score:** 1 / 1

### Question 4

The `touch` command is used to ****\_\_\_****:

- [ ] create a file if it does not exist or otherwise delete it.
- [ ] copy a file or modify its executable permission.
- [x] create a file if it does not exist or otherwise update its "last-modified date" attribute.
- [ ] change the contents of an existing file or its access rights.

**Answer:** Create the file if it doesn't exist or update the file's last-modified timestamp. `touch` is commonly used to create empty files or refresh timestamps.

**Score:** 1 / 1
