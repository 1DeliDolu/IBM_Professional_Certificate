Question 2

Which of the following statements about IBM Cloud Code Engine project and application is correct?


[ ]You are allowed to create only one application for each project.

[ ]You can have two applications with the same name in one project.

[ ]You are allowed to create only one project under your IBM Cloud account.

[X]You can have two applications with the same name in two different projects.

Correct. IBM Cloud Code Engine project provides a namespace, so you only need to keep application names unique within a project. For example, two applications with the name `app1` can be placed into two projects, project1.app1, project2.app1.
