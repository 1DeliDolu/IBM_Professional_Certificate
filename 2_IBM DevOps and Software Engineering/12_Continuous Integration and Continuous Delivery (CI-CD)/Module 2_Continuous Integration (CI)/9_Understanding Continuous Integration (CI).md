Question 2

How do Continuous Integration (CI) tools reduce the reaction time to code changes?

[ ]CI tools reduce the time required to push changes.

[ ]CI tools communicate project updates to clients.

[ ]CI tools fix all code breaks without assistance.

[X]CI tools test changes and builds automatically.

Correct. You can deliver the solution to your client more quickly using a CI tool. When you push a change to a remote branch, the CI tool tests the change automatically, so you don’t have to. Then the tool builds the change and even checks the build for you, saving additional time.
