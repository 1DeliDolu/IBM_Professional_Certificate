[ ]Export the code.

[ ]Shorten the code.

[ ]Modify the code.

Correct. After cloning a new repository for a project on which you will work, you should test the code before doing anything else. Otherwise, if the code doesn’t work later on, you won’t know if you broke the code or if the code was already broken.
