How can you utilize Serverless Reference Architecture for an Event streaming use case? Select two answers.


[ ] You can trigger your application using HTTP events

[ ] Create dependencies on other resources

[X] You can update secondary data stores and caches

Correct. Your event-streaming pipelines can power your analytics systems, update secondary data stores and caches, or feed monitoring systems.
