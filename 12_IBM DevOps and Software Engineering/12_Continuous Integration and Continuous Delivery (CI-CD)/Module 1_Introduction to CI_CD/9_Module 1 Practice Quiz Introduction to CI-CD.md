What is a benefit of Infrastructure as Code (IaC)?

[ ]Greater marketability for potential clients

[ ]Reduced requirements for code readability

[ ]Expanded options for unit and integration testing

[X]Improved protection against staff turnover

Correct. IaC ensures that provisioning intelligence always remains with the organization. To maximize efficiency, only a few skilled engineers or IT staffers typically handle provisioning. If one of these specialists leaves, others are left to reconstruct the process.
