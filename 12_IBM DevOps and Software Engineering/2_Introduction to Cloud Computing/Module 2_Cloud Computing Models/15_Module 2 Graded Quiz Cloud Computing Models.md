[ ]CaaS

[X]SaaS

[ ]Business analytics services

[X]APIs for services

Question 9

[X]Hybrid Cloud

[ ]On-premise Data Centers

[ ]Public Cloud

[ ]Private Cloud

Correct

Correct! With a Hybrid Cloud, organizations can choose to run their mission-critical applications on the private cloud while leveraging additional public cloud capacity to accommodate spikes in demand for their application.

Question 10

What do you mean by “cloud bursting”?

[ ]You can incrementally modernize your core business systems

[X]You can leverage public cloud instances for a period of time but return to the private cloud when the surge is met

[ ]You can spin-up environments using pay-as-you-go in the public cloud

[ ]You can move applications and data between on-premises and cloud systems

Correct

Correct! With proper integration and orchestration between the public and private clouds, you can leverage both clouds for the same workload. You can leverage additional public cloud capacity to accommodate a spike in demand for a private cloud application.
