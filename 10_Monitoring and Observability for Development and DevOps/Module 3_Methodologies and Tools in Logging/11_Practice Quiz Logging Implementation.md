Question 1

Which type of application log is useful for troubleshooting issues, detecting security breaches, and recording user actions?

[ ]Error Logs

[ ]Performance Logs

[ ]Access Logs

[X]Event Logs

Correct! Event logs record application events and user actions. They are useful for troubleshooting issues and detecting security breaches.
